# Calcounter

CALCOUNTER is a web application created to help users track their daily calories and reach their weight goals. Built with React, JavaScript, and CSS.

### Installation

```
$ git clone https://github.com/howardjhkim/calcounter.git
$ cd client
$ npm install
```


### Built With

* React.js

## Upcoming Features
